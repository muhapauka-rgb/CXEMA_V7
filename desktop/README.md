# CXEMA V7 Desktop (Sprint 1)

Этот каталог содержит desktop-оболочку приложения на Electron.

## Что уже реализовано
- Запуск единым desktop-окном `CXEMA V7`.
- Автопроверка локальных сервисов.
- Если backend/frontend не запущены, оболочка поднимает их автоматически.

## Важно
Текущая версия — dev-shell для Спринта 1.
Релизная упаковка `.dmg`, автообновления из GitHub и backup-планировщик добавляются в следующих спринтах.

## Локальный запуск (для разработки)
1. Установить зависимости оболочки:
   - `cd desktop`
   - `npm install`
2. Запустить оболочку:
   - `npm run dev`

## Сборка macOS .dmg (подготовка)
1. Установить зависимости:
   - `cd desktop`
   - `npm install`
2. Запустить упаковку:
   - `npm run pack:mac`
3. Готовый артефакт:
   - `dist-desktop/*.dmg`

## Ожидаемые зависимости на машине разработчика
- `backend/.venv` с установленными Python-пакетами.
- Node/npm и `frontend` зависимости.
