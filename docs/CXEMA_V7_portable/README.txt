CXEMA V7 portable preview

1) Распакуйте папку "site" на другом компьютере.
2) Откройте терминал в папке "site".
3) Запустите локальный сервер:
   - Python 3:  python3 -m http.server 13011
4) Откройте в браузере: http://localhost:13011

Важно:
- Это текущий фронтенд со всеми вкладками и стилями.
- Для живых данных нужен backend API на порту 28011.
- Если backend недоступен, интерфейс откроется, но данные могут не загрузиться.
