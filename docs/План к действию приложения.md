# План к действию приложения

## Цель
Передать CXEMA V7 пользователю без навыков программирования: установка в macOS как обычного приложения, автозапуск всех сервисов, автообновление одной кнопкой, автоматический backup БД в фоне.

## Принципы
- Без терминала и ручных команд для конечного пользователя.
- Все критичные операции в UI: установка, обновление, проверка статуса, восстановление БД.
- Данные пользователя хранятся отдельно от кода приложения.
- Перед обновлением всегда выполняется безопасный backup.

## Архитектура (целевая)
- Desktop shell: Electron (этап 1, без Rust/Tauri зависимости).
- Frontend: React/Vite.
- Backend: FastAPI (локальный процесс внутри desktop app).
- Локальное хранилище пользователя:
  - `~/Library/Application Support/CXEMA/data/app.db`
  - `~/Library/Application Support/CXEMA/backups/app.backup.current.db`
  - `~/Library/Application Support/CXEMA/backups/app.backup.prev.db`

## Backup
- Расписание: ПН/СР/ПТ в 23:00 (локальное время macOS).
- Механика:
  1. `app.backup.current.db` -> `app.backup.prev.db`
  2. Создание нового `app.backup.current.db` из рабочей БД через безопасный SQLite backup.
- Режим: фоновый, без уведомлений.

## Обновления
- Источник: GitHub Releases.
- UI-кнопка в глобальных настройках: `Обновить`.
- Поток:
  1. Проверка новой версии.
  2. Скачивание.
  3. Backup БД.
  4. Применение обновления.
  5. Перезапуск приложения.

## Спринты
1. Desktop shell (запуск backend/frontend из app, одно окно).
2. Перенос БД в Application Support + backup current/prev + восстановление.
3. In-app обновление из GitHub Releases.
4. Onboarding первого запуска и упрощение настроек под non-tech пользователя.
5. Упаковка релиза в `.dmg` + smoke-тесты на "чистой" macOS.

## Риски
- macOS Gatekeeper предупреждения (без Apple Developer notarization).
- Сбой обновления: закрывается backup-before-update.
- Сбой Google OAuth: закрывается экраном статуса + кнопкой переподключения.

## Definition of Done (финальное)
- Пользователь устанавливает приложение через `.dmg`.
- Нажимает и работает без командной строки.
- PDF, Google Drive, импорт/экспорт и все текущие функции доступны.
- Обновление работает из UI.
- БД восстанавливается из backup в один клик.
